CSV format if the line is a project:
1;[project name];[enablers (users) ID "+" separated];[value];[payment date (ddMMyyyy)];[success proba];[status code];[funders (organizations) ID "+" separated];[providers (organizations) ID "+" separated];[revenue type code];[countries alpha3 code "+" separated];[sicknesses code "+" separated];[product code]

Mapping:
  • EnablerID: UserID to be find in Users > User profile
  • Success Probaility: Number from 0 to 100
  • List of status codes: {{status}} : {{status | projectStatus}}
  • FunderID: OrganizationID to be found in Organizations > Organization profile
  • ProviderID: OrganizationID to be found in Organizations > Organization profile
  • List of revenue type codes: {{type}} : {{type | revenueType}}
  • Country: The 3 letters code of a country, e.g. for Belgium = BEL
  • List of sicknesses codes: {{sickness}} : {{sickness | sickness}}
  • List of product codes: {{product}} : {{product | product}}

CSV format if the line is an organization:
2;[organization name];[organization type code]

Mapping:
  • List of organization types codes: {{type}} : {{type | organizationType}}
